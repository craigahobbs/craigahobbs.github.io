# Color Ramp

A tool for creating and editing color gradients. The color ramp library below contains all of Tableau's built-in
gradient color palettes.


## Modern Palettes (Tableau 10.0+, 2016&ndash;present)

Tableau 10.0 (August 2016) shipped a complete color redesign led by color scientist Maureen Stone &mdash; who had
designed the original palettes as a consultant and joined Tableau in late 2011 as a founding member of Tableau Research.
The gradient palettes were rebuilt as densely specified ramps &mdash; 20 stops for sequential palettes and 7 for
diverging &mdash; giving smoother, more perceptually even transitions than the classic two- and three-anchor gradients.
The refresh also introduced new families: gold-anchored warm ramps (Orange-Gold, Green-Gold, Red-Gold, Gold-Purple),
multi-hue ramps (Blue-Teal, Sunset-Sunrise, Red-Green-Gold), a warm neutral (Gray Warm), and white-centered diverging
palettes renamed to the "X-Y-White Diverging" pattern. A few classic palettes carried forward unchanged. Ramps below use
Tableau's full palette definitions.

## Classic Palettes (before Tableau 10.0, 2016)

Tableau's original gradient palettes grew out of Stanford's Polaris research project and the company's first releases in
the mid-2000s. Color scientist Maureen Stone designed Tableau's early color palettes, starting with Tableau 1.5: a small
set of muted, grayed single-hue sequential ramps and gray-centered diverging ramps, several doubling as Tableau's
automatic defaults &mdash; Green for all-positive measures, Red for all-negative, and Red-Green for mixed values. The
roster grew in later classic versions: the "Area" palettes (softer fills for area-style marks), the "Light" pastel
palettes, and several ramps adapted from Cynthia Brewer's published ColorBrewer map schemes &mdash; adaptations of her
published color specifications, not palettes she designed for Tableau. Tableau never published per-palette release
history; the full set below is verifiable as of Tableau 8.0 (2013) and is Tableau's official end-of-era roster. Tableau
10.0 (2016) retired it: most palettes survive under "Classic" names and can be restored via Preferences.tps, five were
dropped outright, and five carried forward unchanged into the modern set. Ramps below use Tableau's official multi-stop
definitions.
